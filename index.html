PhD Research Project

Developed a fast, real-time electromagnetic simulator for coding metasurfaces—ultra-thin 2D arrays of nanoscale structures that control light properties such as phase, polarization, and direction. Leveraged supervised machine learning to accurately predict reflection phase of metasurface unit cells, greatly enhancing design efficiency.

Implemented an active learning optimization strategy that reduced computation time by over 200× compared to traditional genetic algorithms. For example, a 10×10 metasurface array was optimized in 65 minutes instead of 13,260 minutes.

For larger 60×60 arrays, active learning achieved a 24-fold speedup while improving radar cross-section reduction performance. Deep learning was also used to refine analytical models to address mutual coupling effects, further boosting accuracy.

This hybrid machine learning approach established a scalable and precise framework for metasurface design, outperforming existing methods and offering promising advancements in electromagnetics and nanophotonics.

MSc Research Project

The project aimed to detect whistles generated by pressure cookers to ease homemakers’ burden of monitoring cooking and enhance safety by proactive alerts. This was achieved using supervised machine learning for audio classification with a Support Vector Machine (SVM) classifier.

The audio signal, comprising large samples, was processed by extracting representative features from short-term windows (50 ms) and computing statistical representations over mid-term windows (500 ms). These were labeled manually as whistle/no whistle to train the model.

Three feature extraction methods were compared to find a robust model: (a) energy and zero crossing rate features with statistical mid-term features, (b) Mel Frequency Cepstral Coefficients (MFCCs) with mean and standard deviation mid-term features, and (c) a combination of the two.

The trained model was validated on independent recordings and deployed as an Android application called Cybaural Chef, achieving 93% whistle detection accuracy.
